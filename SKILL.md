# IRASUTOYAget — 从 IRASUTOYA 搜索下载插画

## 这个技能做什么

帮用户从 [IRASUTOYA（いらすとや）](https://www.irasutoya.com/) 找到并下载合适的插图。

IRASUTOYA 是日本最大的免费插画素材网站之一，提供大量手绘风格的可爱插图，涵盖人物、动物、食物、商务、教育等各种主题。

## 工作流程

### 第一步：理解用户需求，翻译关键词

用户可能用中文、英文或其他语言描述他们想要的插图。你需要：

1. 理解用户想要什么类型的插图
2. 将关键词翻译成**日语**（IRASUTOYA 是日文网站，日语搜索效果最好）
3. 如果有多个相关日语表达，考虑用哪个最合适

**翻译技巧：**
- 优先用具体名词，而不是抽象描述。比如"开会的人" → 「会議」比「ビジネスマンが集まっている」更好
- 如果关键词有多个日语表达，可以用最常用的那个。比如"猫" → 「猫」（ねこ）
- 对于复合概念，可以尝试拆分成多个搜索词分别搜索

### 第二步：搜索 IRASUTOYA

使用附带的搜索脚本（位于 `<skill-dir>/scripts/search_irasutoya.py`，`<skill-dir>` 即本 SKILL.md 所在目录）：

```bash
python "<skill-dir>/scripts/search_irasutoya.py" "<日语关键词>" --list-only --json --limit 10
```

> **Python 路径提示**：如果 `python` 命令不可用，尝试 `E:/soft/Anaconda/python.exe`。

参数说明：
- `--list-only` 只列出结果不下载
- `--json` 输出 JSON 格式方便分析
- `--limit 10` 限制结果数量

### 第三步：选择最合适的插图

查看搜索结果，根据以下标准选择：

1. **相关性**：标题和标签与用户需求的匹配度
2. **场景**：插图展现的场景是否符合用户用途（比如"开会"而不是"独自办公"）
3. **风格**：如果用户有偏好（可爱、正式、简约等），优先匹配

如果搜索结果不理想，可以：
- 换一个日语关键词重新搜索
- 用更具体或更宽泛的关键词
- 搜索多个相关关键词，分别看看结果

### 第四步：下载插图

确定要哪张图后，下载它：

```bash
python "<skill-dir>/scripts/search_irasutoya.py" "<日语关键词>" --output "<输出目录>" --index <选择的序号>
```

- `--output` 指定保存目录，默认当前目录
- `--index` 指定下载第几个结果（从 0 开始）

下载完成后告诉用户文件保存在哪里。

## 多个关键词的情况

如果用户需求比较复杂（比如"一个在会议室里做演示的商务人士"），可以：

1. 拆成多个关键词分别搜索：「会議室」「プレゼン」「ビジネスマン」
2. 分别展示结果，让用户选择
3. 或者先搜一个核心词，如果结果里有接近的就直接用

## 注意事项

- IRASUTOYA 的插图可以免费用于个人和商业用途，但不要大批量下载或再分发
- 搜索脚本在请求之间会有 0.5 秒的延迟，这是对服务器的礼貌
- 如果搜索没有结果，告诉用户并建议换关键词
- 图片格式取决于源文件，通常为 JPG 或 PNG
